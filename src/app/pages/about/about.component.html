Professional Summary

I'm a passionate Angular Front-End Developer and B.Tech Information Technology graduate from Guru Gobind Singh Indraprastha University (GGSIPU), New Delhi. I specialize in building clean, performant, and scalable web applications using Angular, TypeScript, and modern SCSS architecture.

My journey began with a deep curiosity for how the web works, which evolved into a love for crafting intuitive user interfaces. I believe great software is the intersection of clean code, thoughtful design, and real user needs.

My Experience

During my Front-End Developer Internship at IoTechWorld Avigation, I worked on production Angular applications for IoT dashboards, integrating REST APIs and building reusable component libraries. This gave me hands-on experience in real Agile teams and professional code standards.

My Data Science Internship at YBI Foundation expanded my analytical mindset — working with Python, machine learning models, and data visualization gave me a unique perspective that now informs how I approach data-driven UI decisions.

What I Build

I've developed three substantial projects — a College ERP (Campus Vidyalaya), an Seat Allocation Management System — all built with Angular Standalone Components, Reactive Forms, and clean SCSS architecture. These projects solve real institutional problems.

Beyond Code

Outside of development, I'm deeply interested in Data Structures & Algorithms, competitive problem-solving, and keeping up with the Angular ecosystem. I'm currently seeking challenging roles where I can contribute meaningfully while continuing to grow as an engineer.
